## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-get-availability`
- `allow-get-devices`
- `allow-request-device`
- `allow-connect-gatt`
- `allow-disconnect-gatt`
- `allow-forget-device`
- `allow-get-primary-services`
- `allow-get-characteristics`
- `allow-read-characteristic-value`
- `allow-write-characteristic-value`
- `allow-start-notifications`
- `allow-stop-notifications`
